# إدارة أملاك — النسخة الموحدة

هذه النسخة تجمع الإصلاحات في تحديث واحد:
- إضافة وحذف العقارات.
- إضافة وحذف الوحدات.
- حذف العقار مع وحداته المرتبطة.
- إضافة وحذف المستأجرين والعقود والدفعات والصيانة.
- لوحة تحكم وتقارير مرتبطة بالبيانات الفعلية.
- تحديث حالة الوحدات ونسبة الإشغال تلقائيًا.
- حفظ البيانات محليًا في المتصفح.
