# إدارة أملاك — النسخة الموحدة

نظام عربي محلي لإدارة العقارات والوحدات، يبدأ دون أي بيانات تجريبية.

الوظائف المتاحة:
- إضافة وتعديل وحذف العقارات.
- عمليات إضافة وتعديل وحذف كاملة للعقارات والوحدات والمستأجرين والعقود والدفعات والصيانة.
- علاقات ثابتة بالمعرّفات بين العقار والوحدة والمستأجر والعقد والدفعة وطلب الصيانة.
- حذف متسلسل آمن يعرض للمستخدم عدد السجلات المرتبطة قبل التأكيد.
- لوحة تحكم وتقارير مرتبطة بالبيانات الفعلية.
- تحديث حالة الوحدات ونسبة الإشغال تلقائيًا.
- حفظ البيانات محليًا في المتصفح.
- تحديث مؤشرات لوحة التحكم والتقارير والمخطط والتنبيهات تلقائيًا.
- حفظ إعدادات التنبيهات وإمكانية تعطيل كل نوع منها.
- واجهة متجاوبة للجوال وقائمة وصول إلى جميع الصفحات.

افتح `index.html` في متصفح حديث. تحفظ البيانات في التخزين المحلي للمتصفح نفسه.
