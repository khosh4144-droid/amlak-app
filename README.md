# تطبيق إدارة أملاك — النسخة الأولية

نسخة ويب عربية متجاوبة تعمل بدون أي مكتبات خارجية.

## التشغيل
1. فك الضغط.
2. افتح ملف `index.html` مباشرة في المتصفح.
3. لإتاحة التطبيق على الشبكة المحلية يمكن تشغيل خادم بسيط:
   - Python: `python -m http.server 8000`
   - ثم افتح: `http://localhost:8000`

## الموجود حاليًا
- لوحة تحكم متجاوبة.
- إدارة العقارات وعرضها.
- إضافة عقار جديد مع حفظه في LocalStorage.
- صفحات الوحدات والمستأجرين والعقود والدفعات والصيانة.
- البحث والتصفية.
- دعم الجوال وواجهة RTL عربية.

## المرحلة التالية المقترحة
- تسجيل دخول وصلاحيات متعددة.
- قاعدة بيانات حقيقية.
- API وخادم خلفي.
- إنشاء وتعديل وحذف السجلات.
- رفع العقود والمستندات.
- إشعارات واتساب/SMS.
- تقارير PDF وExcel.
- ربط بوابات الدفع ومنصة إيجار عند توفر التكامل.
