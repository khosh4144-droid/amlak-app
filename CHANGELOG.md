# سجل التغييرات

جميع التغييرات المهمة في نظام «إدارة أملاك» موثقة في هذا الملف.

## 1.0.0 - 2026-07-21

### الإصلاحات

- استكمال عمليات الإضافة والتعديل والحذف للعقارات والوحدات والمستأجرين والعقود والدفعات وطلبات الصيانة.
- تحويل جميع العلاقات الداخلية إلى معرّفات ثابتة: `propertyId` و`unitId` و`tenantId` و`contractId`.
- إضافة ترحيل تلقائي للبيانات القديمة التي كانت تعتمد على الأسماء النصية.
- إضافة تنظيف دفاعي عند التحميل يحذف السجلات التابعة التي تشير إلى عقار أو وحدة أو مستأجر أو عقد غير موجود.
- إضافة حذف متسلسل للعقار والوحدة والمستأجر والعقد مع تنظيف جميع البيانات التابعة.
- عرض عدد الوحدات والمستأجرين والعقود والدفعات وطلبات الصيانة المتأثرة قبل حذف العقار.
- منع إنشاء عقد لا تتطابق وحدته مع وحدة المستأجر.
- منع تغيير وحدة مستأجر مرتبط بعقد قائم دون معالجة العقد أولًا.
- منع حفظ عقد ينتهي قبل تاريخ بدايته.
- إزالة مجموعة المصروفات والحقول المحسوبة والتكرارات غير المستخدمة.
- جعل لوحة التحكم والتقارير والمخطط والتنبيهات تعتمد على البيانات المحفوظة فعليًا.
- تصحيح حساب الدخل عند وجود عقارات بوحدات وعقارات دون وحدات.
- حفظ إعدادات المنشأة والتنبيهات في التخزين المحلي.
- معالجة بيانات التخزين المحلي التالفة وفشل الحفظ عند امتلاء المساحة.
- تقليل إعادة الرسم إلى الصفحة النشطة وتأخير البحث القصير لتحسين الاستجابة.
- تحسين عرض القوائم الكبيرة والبطاقات باستخدام `content-visibility`.
- تحسين تصميم الجوال والنوافذ المنبثقة والوصول إلى جميع الصفحات.
- إضافة حماية للنصوص المعروضة من حقن HTML.

### التحقق والاختبارات

- اختبار CRUD الكامل لجميع الكيانات.
- اختبار حذف العقار مع جميع السجلات التابعة.
- اختبار تنظيف سجلات يتيمة أُدخلت عمدًا إلى التخزين المحلي.
- اختبار إعادة تحميل الصفحة واستعادة البيانات والإعدادات.
- اختبار ضغط يتضمن 100 عقار و1000 وحدة و1000 مستأجر و1000 عقد و1000 دفعة و1000 طلب صيانة.
- اختبار واجهة الجوال بقياس 390 × 844.
- فحص صياغة JavaScript ومعرّفات HTML وربط جميع الأزرار.

### حدود الإصدار

- البيانات محلية في المتصفح ولا تتم مزامنتها بين الأجهزة.
- لا توجد حسابات مستخدمين أو صلاحيات أو قاعدة بيانات خادمية.
- مسح بيانات المتصفح يؤدي إلى حذف بيانات النظام ما لم تُنسخ خارجيًا.
